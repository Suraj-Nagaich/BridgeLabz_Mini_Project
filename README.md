# Student Management System (Mini Project)

## Overview

This project is a Student Management System built using HTML, CSS, JavaScript, AJAX, and JSON Server.
It supports Create, Read, Update, Delete (CRUD) operations for student data stored on a JSON Server.

## Features
- Add new students
- View list of students
- Update student details
- Delete students
- Async operations using Fetch API
- Clean UI with HTML & CSS
- Fully dynamic using JS classes and JSON Server

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)
- JSON Server (Fake REST API)

## How to Run the Project
### Step 1: Install JSON Server
npm install -g json-server

### Step 2: Start JSON Server
json-server --watch db.json --port 3000

### Step 3: Open index.html in browser
